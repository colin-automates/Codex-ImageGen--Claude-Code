# Code of Conduct

This project follows the **Contributor Covenant**, version 2.1.

The full text is published at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In short: be respectful, assume good faith, focus on the work, and help maintainers keep the project welcoming to people from all backgrounds. Reports of behavior that violates the Covenant can be sent via a GitHub Issue marked `code-of-conduct` or by emailing the maintainer listed in the repo.
